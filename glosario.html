Vocabulario web

Glosario

# Inglés Español Significado
01 User Interface (UI) Interfaz de usuario Conjunto de elementos visuales e interactivos que permiten al usuario comunicarse con un sistema digital, como botones, menús y formularios.
02 User Experience (UX) Experiencia de usuario Percepción global que tiene una persona al interactuar con un producto digital, considerando usabilidad, emociones y eficiencia.
03 Wireframe Esquema de pantalla Boceto de baja fidelidad que representa la estructura y distribución de los elementos en una interfaz, sin colores ni estilos definitivos.
04 Responsive Design Diseño adaptable Técnica de desarrollo web que permite que una página se adapte y se vea correctamente en distintos tamaños de pantalla, desde móviles hasta escritorios.
05 Accessibility Accesibilidad Práctica de diseñar interfaces que puedan ser usadas por personas con distintas capacidades, incluyendo quienes usan tecnologías de asistencia.
06 Prototype Prototipo Versión simulada de un producto digital que permite probar flujos e interacciones antes de escribir código, reduciendo errores en el desarrollo.
07 Component Componente Bloque de interfaz reutilizable y autónomo, como un botón o una tarjeta, que puede combinarse para construir páginas completas de manera consistente.
08 Grid System Sistema de rejilla Estructura de columnas y filas invisible que organiza y alinea los elementos de una página para lograr un diseño ordenado y coherente.
09 Typography Tipografía Selección y uso de fuentes tipográficas en una interfaz para comunicar jerarquía, personalidad de marca y facilitar la lectura del contenido.
10 Call to Action (CTA) Llamada a la acción Elemento de interfaz, normalmente un botón o enlace, que invita al usuario a realizar una acción específica como registrarse, comprar o descargar.
11 Hover State Estado de desplazamiento Cambio visual que ocurre en un elemento interactivo cuando el cursor del usuario se posiciona sobre él, indicando que es clicable o activo.
12 Loading State Estado de carga Indicador visual que informa al usuario que el sistema está procesando una acción, reduciendo la incertidumbre durante tiempos de espera.
13 Whitespace Espacio en blanco Área vacía entre los elementos de una interfaz que mejora la legibilidad, reduce la carga visual y guía la atención del usuario hacia lo importante.
14 Visual Hierarchy Jerarquía visual Organización de los elementos de una página según su importancia, usando tamaño, color y posición para guiar la mirada del usuario de forma natural.
15 Usability Testing Prueba de usabilidad Evaluación en la que usuarios reales interactúan con un producto digital para detectar problemas de diseño y validar si la interfaz cumple sus objetivos.
